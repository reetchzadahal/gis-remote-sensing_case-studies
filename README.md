# GIS & Remote Sensing Case Studies

## Overview

This repository is a collection of case studies and analytical projects focused on GIS and Remote Sensing applications in environmental monitoring, agriculture, and climate-related challenges.

The work primarily involves the use of geospatial data, satellite imagery, and spatial analysis techniques to understand real-world problems such as drought, land degradation, and food security.

---

## Case Studies Included

### 1. Agricultural Drought Assessment using VCI (FAO)

* Analysis of agricultural drought using satellite-based Vegetation Condition Index (VCI)
* Assessment of drought exposure in rainfed and irrigated agriculture
* Comparative study across multiple years
* Application for food security and climate resilience
